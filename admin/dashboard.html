Media uploads

Upload images, videos, or PDFs to use in your products and blog posts. Files download to your computer — save them into assets/images/ or assets/files/ in your site folder.

Upload media

How to use uploaded files

  1. After downloading the optimized file, drop it into your site's assets/images/ folder (for images) or assets/files/ folder (for PDFs and videos).
  2. Reference it in a product or blog post using the path, e.g. assets/images/my-photo.jpg.
  3. Re-upload your site to your host (Netlify, Vercel, GitHub Pages, or wherever it lives).

Export & deploy

Export your edited JSON files. Replace the matching files in your site's /data/ folder, then redeploy.

Download updated data

Deploy workflow

  1. Make all edits in the admin (changes save in your browser).
  2. Click Export on each section.
  3. Replace the matching files inside your site's /data/ folder.
  4. If your site is on Netlify, drag-and-drop the updated folder onto your Netlify dashboard. On Vercel/GitHub Pages, commit and push.
  5. The site rebuilds in seconds.

Pro tip: If you connect this folder to a GitHub repo and enable Netlify CMS (Decap CMS), edits will publish automatically — see README.
